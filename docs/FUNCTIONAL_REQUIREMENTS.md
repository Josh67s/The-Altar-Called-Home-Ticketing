# Functional Requirements Specification (FRS)

Amazing Crew TV Production

Project:
Event Ticketing & Event Management Platform

First Event:
THE ALTAR CALLED HOME

## Module 1 – Landing Page

The system shall:

- Display the movie poster.
- Display the movie title.
- Display the movie synopsis.
- Display the event date.
- Display the event time.
- Display the venue.
- Display the official movie trailer.
- Display ticket categories.
- Display ticket prices.
- Provide a "Buy Tickets" button.

## Module 2 – Ticket Selection

The system shall:

- Allow the user to select multiple ticket categories.
- Allow the user to select ticket quantities.
- Calculate the total amount automatically.
- Display a shopping cart summary.
- Allow the user to edit ticket quantities.

## Module 3 – Buyer Information

The system shall:

- Collect buyer's full name.
- Collect buyer's email address.
- Collect buyer's phone number.
- Validate required fields.

## Module 4 – Attendee Information

The system shall:

- Generate attendee forms automatically based on the number of tickets purchased.
- Collect attendee names.
- Collect attendee email addresses.
- Collect attendee phone numbers.
- Assign the correct ticket category to each attendee.

## Module 5 – Payment

The system shall:

- Integrate with Paystack.
- Calculate the payment amount.
- Process secure online payments.
- Verify successful payments.
- Prevent duplicate payment processing.

## Module 6 – Ticket Generation

The system shall:

- Generate a unique ticket ID.
- Generate a QR code.
- Assign the correct ticket category.
- Assign the correct seating zone.
- Generate a digital ticket.

## Module 7 – Email

The system shall:

- Email tickets automatically.
- Email one ticket per attendee.
- Email the buyer a purchase confirmation.

## Module 8 – Admin Dashboard

The system shall:

- Display ticket sales.
- Display revenue.
- Display attendee statistics.
- Display ticket category statistics.
- Allow ticket searching.
- Allow order searching.

## Module 9 – Check-in

The system shall:

- Scan QR codes.
- Verify ticket validity.
- Mark attendees as checked in.
- Prevent duplicate check-ins.

## Module 10 – Reports

The system shall:

- Export attendee lists.
- Export ticket sales.
- Export revenue reports.
- Display event statistics.
