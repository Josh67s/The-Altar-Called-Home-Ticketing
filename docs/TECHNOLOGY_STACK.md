## Coding Standards

- Use meaningful variable names.
- Keep functions small and focused.
- Reuse code where possible.
- Comment only where necessary.
- Avoid duplicate code.
- Keep files organized by responsibility.

## Security Principles

- Never expose secret API keys.
- Validate payments on the server.
- Validate all user input.
- Restrict database access with Firestore Security Rules.
- Log important administrative actions.
